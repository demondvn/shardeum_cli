# shardeum_cli

## Install
    curl -s https://deb.nodesource.com/setup_16.x | sudo bash
    apt install nodejs -y
    git clone https://github.com/demondvn/shardeum_cli.git 
    cd shardeum_cli 
    npm i typescript -g
    npm i 
    npm run compile
    npm i -g .
    npm link
## Auto stake
    operator-cli stake_multi 10.1 <wallets.json> <backup folder>
